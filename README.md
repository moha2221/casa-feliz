# Casa Feliz Website

Statische One-Page-Website für die Ferienwohnung Casa Feliz in Torrox Costa.

## 📁 Dateien

- `index.html` – Website
- `styles.css` – Design
- `assets/` – Bilder (hero.jpg, bedroom.jpg, kitchen.jpg, living.jpg, dining.jpg, bath.jpg, pool.jpg)

## 🚀 Nutzung

### Lokal testen
Einfach `index.html` im Browser öffnen.

### Online veröffentlichen
Alle Dateien bei einem Webhoster (z.B. Netlify, Vercel, GitHub Pages) hochladen:

1. `index.html`
2. `styles.css`
3. `assets/` Ordner mit allen Bildern

## 🎨 Design

- Modernes, responsives Design
- Mobile-freundlich
- Mediterrane Farbpalette (Olivgrün, Terrakotta, Sand)
- Optimiert für alle Bildschirmgrößen

## 📝 Inhalte

- Hero-Sektion mit Balkonbild
- Wohnungsausstattung mit Bildern
- Preisliste (saisonal)
- Galerie
- Umgebung & Ausflüge
- Pool & Lage
- Kontaktbereich
